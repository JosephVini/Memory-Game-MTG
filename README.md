# Memory-Game-MTG

Olá, esse é um projeto inspirado no meu jogo favorito, Magic: The Garthering.
É um jogo da memoria em que o player tem que achar os pares de cada carta.

## Tecnologias Utilizadas:
* HTML
* CSS
* JavaScript
